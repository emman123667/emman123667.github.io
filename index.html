Projects

NoteThemAll - A revision app [A-level]

A web app that I wrote for my A-level Computer Science programming project using MySQL, Vanilla PHP and JavaScript. Allows users to create and manage revision notes digitally, and it also includes a login and registration system. Also features a file upload system to attach videos, audio and images to notes, as well as an interactive quiz system to help users test themselves on a topic they are studying with existing notes.

This is the first full app that I made that features a Graphical User Interface (GUI) and server-side code. A remastered version of this app is planned.
